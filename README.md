# 坦克大战 (Tank Battle)

一个基于 HTML5 Canvas 的经典坦克大战游戏。

## 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 游戏操作

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格键` | 射击 |

## 游戏目标

- 消灭所有敌方坦克
- 保护你的基地（粉色方块）不被摧毁
- 通过所有关卡

## 游戏元素

- **绿色坦克**：玩家
- **红色坦克**：敌人
- **棕色方块**：砖墙（可摧毁）
- **灰色方块**：钢墙（不可摧毁）
- **蓝色方块**：水域（不可通过）
- **绿色半透明**：草丛（可隐藏）
- **粉色方块**：基地（必须保护）

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- 无第三方依赖

## 本地运行

```bash
git clone https://github.com/你的用户名/tank-battle.git
cd tank-battle
# 直接双击 index.html 或在浏览器中打开
```
